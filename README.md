# SistemaiAcademy
Sistema iAcademy desenvolvido por Julia Barros - <a href="https://www.linkedin.com/in/juliabarrosneves/">Linkedin</a> - para o Trabalho de Conclusão de Curso - TCC/Projeto Final do curso de Análise e Desenvolvimento de Sistema. É um sistema para gestão de academia com o objetivo de informatização e facilidade para os serviços administrativos e com plataforma para integração do professor com o aluno, com montagem e visualização de treinos online.
